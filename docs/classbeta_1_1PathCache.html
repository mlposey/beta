Detailed Description

Holds a collection of queryable paths in memory.

PathCache contains a collection of shareable paths. I.e., given a node origin and destination, the cache will look for paths that contain a minimal route that answers the query.

The cache is memory sensitive and will prune suboptimal paths from the collection if a capacity is reached.
