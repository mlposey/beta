size_t beta::Config::getBytes ( std::string  key)

Gets the configuration value for a key that maps to a storage size.

A storage size is a string type that contains a positive number followed by a unit. Units may be not included - indicates the unit is bytes kB - kilobytes MB - megabytes GB - gigabytes Examples: "100" = 100 bytes "250MB" = 250 megabytes

Returns
the size converted to bytes or 0 if the key is unrecognized
